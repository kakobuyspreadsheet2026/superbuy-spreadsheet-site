From grid to cart: a sane sequence

  1. Start on the home grid — open the spreadsheet home, scroll, and let rows load; or start with search / image search if you already have a clue.
  2. Open the listing — click a card; MaisonLooks should open in a new tab with variants and seller copy.
  3. Read carefully — confirm measurements, color codes, batch language, and whether stock is live or pre-order.
  4. Copy what PonyBuy expects — full URL, item ID, or notes—follow their submission form, not a chat screenshot.
  5. Pay on the agent — use ponybuy.cc (logged-in balance or checkout); ignore payment requests in unsolicited messages.
  6. QC before international ship — zoom warehouse photos; ask for retakes while return windows still exist.
  7. Pick a freight line & declare honestly — country rules differ; lean on PonyBuy docs plus your local experience.

What to double-check before you pay

  • Returns & cancellations — policies differ by seller and agent; read while exit is cheap.
  • Measurements — convert units; factory charts rarely match mall sizing.
  • Batch chatter — hype may reference an older run; verify on the live listing.
  • Declarations — you own truthful customs paperwork.
  • Landed math — item + fees + freight + possible VAT—sum it before you top up.
Safety tip. Phishing loves agent keywords. Bookmark ponybuy.cc from a source you trust; never trust a “shortcut” spreadsheet that asks for passwords.

Easy-to-fix pitfalls

  • Letting chats set your risk tolerance — verify on the listing and with PonyBuy.
  • Ignoring return windows — they shrink once outbound shipping starts.
  • Mixing up color names — “black” and “off-black” are different arguments.
  • Underestimating holidays — China’s calendar moves your timeline.
  • Bookmarking phishing domains — verify ponybuy.cc character by character.

Rough timeline (order → door)

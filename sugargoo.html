What is Sugargoo (the short version)

Sugargoo is not a marketplace storefront in the Amazon sense—it is a service layer on top of Chinese marketplaces you cannot easily pay directly.

How to use this site, then buy on Sugargoo (or any agent)

  1. Start on the home grid — open the spreadsheet home, scroll, and let rows load; or start with search / image search if you already have a clue.
  2. Open the listing — click a card; MaisonLooks should open in a new tab with variants and seller copy.
  3. Read carefully — confirm measurements, color codes, batch language, and whether stock is live or pre-order.
  4. Copy what Sugargoo expects — full URL, item ID, or notes—follow their submission form, not a chat screenshot.
  5. Pay on the agent — use sugargoo.com (logged-in balance or checkout); ignore payment requests in unsolicited messages.
  6. QC before international ship — zoom warehouse photos; ask for retakes while return windows still exist.
  7. Pick a freight line & declare honestly — country rules differ; lean on Sugargoo docs plus your local experience.

What to double-check before you pay

  • Seller notes — read the fine print on pre-sale, processing time, and color variance.
  • Size tolerance — measure twice; factory tolerances are not boutique tolerances.
  • Version drift — “same link” can mean different factory weeks; ask if unsure.
  • Insurance — decide whether you need add-ons before the parcel leaves China.
  • Payment timing — know what triggers non-refundable charges on your agent.
Safety tip. Phishing loves agent keywords. Bookmark sugargoo.com from a source you trust; never trust a “shortcut” spreadsheet that asks for passwords.
